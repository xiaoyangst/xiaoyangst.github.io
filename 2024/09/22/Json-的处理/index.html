Json 的处理
#Qt 2024-09-22
  • 创建JSON
  • 解析JSON

在 Qt 中处理 JSON 数据时，通常会使用以下头文件：

  1. **QJsonDocument**：用于封装 JSON 文档（可以是 JSON 对象或 JSON 数组）
  2. **QJsonObject**：用于表示 JSON 对象（key-value 形式的 JSON 数据）
  3. **QJsonArray**：用于表示 JSON 数组
  4. **QJsonValue**：用于表示 JSON 对象或数组中的值（可以是字符串、数字、布尔值等）
  5. **QJsonParseError**：用于捕捉 JSON 解析中的错误

序列化和反序列化 JSON 用 QJsonDocument，创建 JSON 用 QJsonObject。

想要在网络中进行传输，需要把 QJsonObject 转换为 QByteArray。
